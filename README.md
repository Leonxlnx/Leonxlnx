## Taste Skill

Taste Skill is a small, practical skill system for frontend taste: less generic AI output, more intentional interface design.

It focuses on the details that make AI-built products feel real: layout decisions, type scale, spacing rhythm, interaction polish, visual hierarchy, and motion direction.
